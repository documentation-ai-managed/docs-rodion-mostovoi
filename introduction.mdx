## Quick Start

Get up and running in minutes. Follow these steps to create your first project.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://dashboard.example.com` and create an account with your email.

    ```bash
    curl -X POST https://api.example.com/v1/auth/signup \
      -H "Content-Type: application/json" \
      -d '{"email": "user@example.com", "password": "securepass"}'
    ```
  </Step>
  <Step title="Create a Project" icon="plus">
    Navigate to the dashboard and click "New Project". Name it and set a description.
  </Step>
  <Step title="Add Your First Task" icon="check-circle">
    Inside the project, add tasks with due dates and assignees.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      const response = await fetch('https://api.example.com/v1/tasks', {
        method: 'POST',
        headers: { 'Authorization': `Bearer ${YOUR_API_KEY}`, 'Content-Type': 'application/json' },
        body: JSON.stringify({
          title: 'Complete documentation',
          projectId: 'proj_123',
          dueDate: '2024-12-31'
        })
      });
      ```
      ```python
      import requests
      response = requests.post(
        'https://api.example.com/v1/tasks',
        headers={'Authorization': f'Bearer {YOUR_API_KEY}', 'Content-Type': 'application/json'},
        json={'title': 'Complete documentation', 'projectId': 'proj_123', 'dueDate': '2024-12-31'}
      )
      ```
    </CodeGroup>
  </Step>
</Steps>
