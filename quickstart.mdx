## Prerequisites

Before starting, ensure you have:

- A valid email address
- Access to your email for verification
- A modern web browser like Chrome or Firefox

<Callout kind="info">

Prepare your `YOUR_API_KEY` from the dashboard after registration. You'll need it for API calls.

</Callout>

## Register Your Account

Follow these steps to create your Rodion Mostovoi account.

<Steps>
  <Step title="Visit Registration Page" icon="user-plus">
    Navigate to `https://dashboard.example.com/register`.
  </Step>
  <Step title="Enter Details" icon="edit-3">
    Provide your email, name, and password. Use a strong password.
  </Step>
  <Step title="Verify Email" icon="mail">
    Check your inbox and click the verification link sent to your email.
  </Step>
  <Step title="Complete Setup" icon="check-circle">
    Return to the site and set up your profile preferences.
  </Step>
</Steps>

## Create Your First Task

Use the dashboard or API to add a task.

### Dashboard Method

1. Click **New Task** in the sidebar.
2. Enter title: "Complete Quickstart".
3. Set due date and priority.
4. Save.

### API Method

Authenticate with your `YOUR_API_KEY`.

<CodeGroup tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/tasks', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer YOUR_API_KEY`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      title: 'Complete Quickstart',
      description: 'Follow the guide to get started.',
      due_date: '2024-12-31'
    })
  });

  const task = await response.json();
  console.log(task);
  ```
  ```bash
  curl -X POST https://api.example.com/v1/tasks \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "title": "Complete Quickstart",
      "description": "Follow the guide to get started.",
      "due_date": "2024-12-31"
    }'
  ```
</CodeGroup>

<Callout kind="success">

Your first task is created! Check the dashboard to view it.

</Callout>
