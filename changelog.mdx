## Detailed Changes

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time dashboard updates using WebSockets at `wss://dashboard.example.com/ws`.
- Introduced customizable themes with your brand color `#3B82F6`.

## Improvements

- Optimized API response times by `>30%` for endpoints like `https://api.example.com/users`.
- Enhanced error handling with detailed logs.

## Bug Fixes

- Fixed intermittent connection drops in long-running sessions.
- Resolved CSS conflicts in dark mode.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## New Features

- Core API launch with user management at `https://api.example.com/users`.

## Upgrading to the Latest Version

<Callout kind="info">
  Review migration notes before upgrading. Test in staging first.
</Callout>

<Steps>
  <Step title="Backup Data" icon="database">
    Export your configurations and data.
  </Step>
  <Step title="Update Package" icon="package">
    
````bash
npm install rodion-mostovoi@latest
````

  </Step>
  <Step title="Migrate Config" icon="settings">
    
<CodeGroup tabs="JavaScript,Python">
```javascript
// Old config
const config = { apiKey: 'old_key' };

// New config
const config = { 
  apiKey: 'YOUR_API_KEY',
  themeColor: '#3B82F6' 
};
```
```python
# Old config
config = {'api_key': 'old_key'}

# New config
config = {
    'api_key': 'YOUR_API_KEY',
    'theme_color': '#3B82F6'
}
```
</CodeGroup>

  </Step>
  <Step title="Test Endpoints" icon="check-circle">
    Verify `https://api.example.com/v1/users` returns expected data.
  </Step>
</Steps>
